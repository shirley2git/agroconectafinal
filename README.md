## 💡 Sobre o Projeto

O **AgroConecta** é uma plataforma digital desenvolvida para ser a **solução tecnológica para o pequeno agricultor familiar brasileiro**. O projeto nasceu como um **PBL (Project Based Learning)** na FIAP e está evoluindo de uma interface estática para uma aplicação web moderna e escalável.

Nosso foco é a **Agenda ODS 2 da ONU (Fome Zero e Agricultura Sustentável)**, facilitando o acesso a dados climáticos, gestão de insumos e educação técnica.

---

## 🚀 Evolução Tech: Do HTML ao React

O projeto passou por uma refatoração completa para adotar as melhores práticas do ecossistema **Front-end moderno**:

* **Componentização:** Divisão da interface em componentes reutilizáveis (Header, Footer, Form, Card).
* **Gerenciamento de Estado:** Uso de `useState` e `useEffect` para controlar a calculadora e as validações em tempo real.
* **Tipagem com TypeScript:** Maior segurança no desenvolvimento e redução de bugs em tempo de execução.
* **Performance:** Migração do carregamento tradicional para o **Vite**, garantindo um HMR (Hot Module Replacement) extremamente rápido.

## ✨ Funcionalidades Dinâmicas

### 1. Calculadora de Insumos (React Version) 🧮
Agora implementada como um componente funcional, a calculadora utiliza estados para processar os cálculos instantaneamente conforme o usuário digita.
* **Lógica:** Cálculo automatizado de `Área (ha) * Insumo` com retorno visual imediato.

### 2. Form de Contato com Validação Customizada 📧
Refatorado para utilizar lógica de hooks, garantindo que o agricultor preencha corretamente nome, sobrenome e contatos válidos antes do envio.

### 3. Assistente Virtual "Dona Rosa" 🤖
Interface preparada para integração futura com APIs de IA, focada na Persona do Sr. José de Almeida.
